用法 使用 javascript 创建流程条（step）

        <div id="demo"></div>

        var opt = {
            "item": [{
                "text": "第一步"
            }, {
                "text": "第二步"
            }, {
                "text": "第三步"
            }],
            step: 2,
            visited: true,
            afterStep:lalala
        };
        var dd = $("#demo").AIStep(opt);

属性

名称 类型 描述
itemarray流程条内容
textstring流程条文本
stepnumber默认进度
visitedboolean是否可以返回

事件

名称 参数 描述
afterStepfunction切换流程条事件调用的方法名称

方法

名称 参数 描述
prevStepnone上一步
nextStepnone下一步
getStepnone获取当前步数
