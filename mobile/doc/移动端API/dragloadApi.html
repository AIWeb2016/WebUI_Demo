移动端-下拉刷新上拉翻页加载API

 webui.css、jquery.js、webui.js
				 

<link rel="stylesheet" type="text/css" href="css/webui.css" />

<script src="jquery.js" type="text/javascript"></script>

<script src="webui.js" type="text/javascript"></script>
				
<div id="inner" class="inner">
	<div class="lists">
		····
	</div>
</div>
 

$('#inner').drag({
	refreshURL:'json/update.json',//刷新的url
	loadMoreURL:'json/more.json',//加载的url
	loadMoreText:'显示下面20条',//底部提示文字
	model:function(){}//模板函数
});	

					

refreshURL:刷新列表所要访问的地址

loadMoreURL:加载更多所要访问的地址

loadMoreText:底部按钮显示的文字，如果为空则不现实按钮

model:页面列表填充的模板，可以自定义
